# MediTrack – Patient & Appointment Handling System
## Project Overview
MediTrack is a Java-based console application developed to manage essential hospital activities efficiently. It focuses on handling patient records, doctor details, and appointment scheduling in a structured manner.

This project is built using core Java and Object-Oriented Programming concepts with a menu-driven system for smooth interaction.

---

## Features
- Add new patients
- Add doctor information
- Schedule appointments
- View patient records
- View doctor details
- View appointment history

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

---

## Project Structure
- **Patient Class** – Stores patient details such as ID, name, age, and gender  
- **Doctor Class** – Stores doctor information such as ID, name, and specialty  
- **Appointment Class** – Manages appointment details between patients and doctors  
- **HospitalManagement Class** – Controls the main workflow of the system  

---

## How It Works
1. User selects an option from the menu  
2. Data is entered through the console  
3. Information is stored using ArrayLists  
4. Appointments are linked through patient and doctor IDs  
5. Records can be viewed anytime
